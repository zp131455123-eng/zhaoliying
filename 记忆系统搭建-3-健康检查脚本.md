# 记忆系统提示词 (3/4) — 11 个健康检查脚本

## 系统健康检查脚本集

### run-all.sh (主入口)

```bash
#!/bin/bash
SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"
TOTAL=0; PASSED=0; FAILED=0

echo "🔍 系统健康检查 - $(date '+%Y-%m-%d %H:%M:%S')"

for CHECK in "$SCRIPT_DIR"/[0-9]*.sh; do
  [ -f "$CHECK" ] || continue
  NAME=$(basename "$CHECK" | sed 's/^[0-9]*-//' | sed 's/.sh$//')
  TOTAL=$((TOTAL + 1))
  
  if "$CHECK" >/dev/null 2>&1; then
    echo "✅ $NAME"
    PASSED=$((PASSED + 1))
  else
    echo "❌ $NAME"
    FAILED=$((FAILED + 1))
  fi
done

echo "结果: $PASSED/$TOTAL 通过"
[ "$FAILED" -gt 0 ] && exit 1 || exit 0
```

#### 02-memory-size.sh — MEMORY.md 行数检查

```bash
#!/bin/bash
MEMORY="$HOME/.openclaw/workspace/MEMORY.md"
[ -f "$MEMORY" ] || exit 1

LINES=$(wc -l < "$MEMORY")

if [ "$LINES" -gt 200 ]; then
  echo "🚨 紧急：MEMORY.md 超过 200 行 (当前 $LINES)" >&2
  exit 1
elif [ "$LINES" -gt 150 ]; then
  echo "⚠️ 警告：MEMORY.md 超过 150 行 (当前 $LINES)" >&2
  exit 1
fi

exit 0
```

**检查**：
- >200 行 → 紧急（退出码 1）
- >150 行 → 警告（退出码 1）

#### 03-sync-freshness.sh — 项目同步过期检查

```bash
#!/bin/bash
WORKSPACE="$HOME/.openclaw/workspace"
NOW=$(date +%s)
STALE_THRESHOLD=7200  # 2 小时 = 7200 秒

for SYNC_DIR in "$WORKSPACE/memory"/*-sync; do
  [ -d "$SYNC_DIR" ] || continue
  SYNC_NAME=$(basename "$SYNC_DIR")
  TS_FILE="$SYNC_DIR/.last-sync-ts"
  
  if [ -f "$TS_FILE" ]; then
    LAST_SYNC=$(cat "$TS_FILE")
    AGE=$((NOW - LAST_SYNC))
    
    if [ "$AGE" -gt "$STALE_THRESHOLD" ]; then
      echo "⚠️ $SYNC_NAME 已过期 ${AGE}秒 (>2h)" >&2
      exit 1
    fi
  else
    echo "⚠️ $SYNC_NAME 缺少同步时间戳" >&2
    exit 1
  fi
done

exit 0
```

**检查**：遍历 `*-sync/.last-sync-ts`，>2h 报警

#### 05-tools-version.sh — 工具版本 vs TOOLS.md 记录

```bash
#!/bin/bash
```

#### 06-log-rotation.sh — 日志归档

```bash
#!/bin/bash
LOG_DIR="$HOME/.openclaw/workspace/memory/logs"
ARCHIVE_DIR="$HOME/.openclaw/workspace/memory/archive/logs"
MAX_LOGS=20

[ -d "$LOG_DIR" ] || exit 0
mkdir -p "$ARCHIVE_DIR"

LOG_COUNT=$(find "$LOG_DIR" -name "*.md" | wc -l)

if [ "$LOG_COUNT" -gt "$MAX_LOGS" ]; then
  echo "📦 日志过多 ($LOG_COUNT/$MAX_LOGS)，归档最旧的 3 个"
  
  find "$LOG_DIR" -name "*.md" -type f | sort | head -n 3 | while read -r LOG; do
    mv "$LOG" "$ARCHIVE_DIR/"
    echo "  归档: $(basename "$LOG")"
  done
fi

exit 0
```

**检查**：日志 >20 个时，自动归档最旧的 3 个到 `archive/logs/`

#### 07-doc-drift.sh — 文档漂移检查

```bash
#!/bin/bash
# 检查过去 24h 有代码提交但没改文档
WORKSPACE="$HOME/.openclaw/workspace"

for PROJECT in /path/to/project-a /path/to/project-b; do
  [ -d "$PROJECT/.git" ] || continue
  
  cd "$PROJECT"
  PROJ_NAME=$(basename "$PROJECT")
  
  # 最近 24h 的 commit
  CODE_COMMITS=$(git log --since="24 hours ago" --oneline -- '*.py' '*.js' '*.ts' | wc -l)
  DOC_COMMITS=$(git log --since="24 hours ago" --oneline -- 'docs/' 'README.md' | wc -l)
  
  if [ "$CODE_COMMITS" -gt 0 ] && [ "$DOC_COMMITS" -eq 0 ]; then
    echo "⚠️ $PROJ_NAME: 有代码提交但无文档更新 (24h 内)" >&2
    exit 1
  fi
done

exit 0
```

**检查**：过去 24h 有代码提交但没改文档？

#### 08-gateway-memory.sh — Gateway 内存检查

```bash
#!/bin/bash
# 检查 OpenClaw Gateway 进程内存占用
GATEWAY_PID=$(pgrep -f "openclaw.*gateway" | head -n 1)

[ -z "$GATEWAY_PID" ] && exit 0

if [ "$(uname)" = "Darwin" ]; then
  # macOS
  MEM_MB=$(ps -o rss= -p "$GATEWAY_PID" | awk '{print int($1/1024)}')
else
  # Linux
  MEM_MB=$(ps -o rss= -p "$GATEWAY_PID" | awk '{print int($1/1024)}')
fi

if [ "$MEM_MB" -gt 10240 ]; then
  echo "🚨 Gateway 内存过高: ${MEM_MB}MB (>10GB)" >&2
  exit 1
elif [ "$MEM_MB" -gt 5120 ]; then
  echo "⚠️ Gateway 内存偏高: ${MEM_MB}MB (>5GB)" >&2
  exit 1
fi

exit 0
```

**检查**：
- >10GB → 紧急
- >5GB → 警告

#### 10-disk-space.sh — 磁盘空间检查

```bash
#!/bin/bash

  exit 1
fi

exit 0
```

**检查**：
- >90% → 紧急

#### 11-network.sh — 网络连通性检查

```bash
#!/bin/bash
# 测试关键域名连通性
DOMAINS=(
  "api.openai.com"
  "api.anthropic.com"
  "dashscope.aliyuncs.com"
)

FAILED=0

for DOMAIN in "${DOMAINS[@]}"; do
  if ! curl -sf --max-time 5 "https://$DOMAIN" >/dev/null 2>&1; then
    echo "❌ 无法连接: $DOMAIN" >&2
    FAILED=$((FAILED + 1))
  fi
done

[ "$FAILED" -gt 0 ] && exit 1 || exit 0
```

**检查**：curl 测试关键域名连通性

### Cron 配置

每天 08:00 自动运行，结果写入 `memory/health-check-latest.md`：

```bash
openclaw cron add --name "daily-health-check" \
  --schedule "0 8 * * *" \
  --command "bash ~/.openclaw/workspace/scripts/health-checks/run-all.sh > ~/.openclaw/workspace/memory/health-check-latest.md"
```
