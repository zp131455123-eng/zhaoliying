**时间**: 22:30 - 22:57 (UTC+8)

- 原因：模型名称错误（用了 text-embedding-v2）

**解决过程**：
1. ✅ 创建测试脚本 `test-dashscope.ps1`
2. ✅ 测试多个模型名称（v1, v2, v3, ada-002）
3. ✅ 确认正确配置：`text-embedding-v3` (dimension: 1024)
4. ✅ 更新 openclaw.json，添加完整的 memorySearch 配置
5. ✅ Gateway 重启（强制 kill node 进程）
6. ✅ 语义搜索功能启用（mode: hybrid）

**最终配置**：
```json
{
  "memorySearch": {
    "enabled": true,
    "provider": "openai",
    "model": "text-embedding-v3",
    "remote": {
      "baseUrl": "https://dashscope.aliyuncs.com/compatible-mode/v1",
      "apiKey": "${DASHSCOPE_API_KEY}",
      "proxy": false
    },
    "query": {
      "hybrid": {
        "enabled": true,
        "vectorWeight": 0.7,
        "textWeight": 0.3
      }
    }
  }
}
```

- ✅ 语义搜索已启用（hybrid 模式）
