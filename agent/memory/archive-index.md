# Archive Index - 归档索引

## 归档规则

**何时归档**：
- 项目已完结超过 7 天
- MEMORY.md 超过 150 行
- "最近学习/发现"超过 7 天的条目

**归档位置**：
- 项目档案：`memory/archive/projects/`
- 日志归档：`memory/archive/logs/`
- 通用归档：`memory/archive/`

**如何查找**：
- 使用 memory_search 语义搜索（归档文件也会被索引）
- 查看本索引文件
